## DFD

일정 조회

~~~
(전체일자)
토큰으로 유저 객체 조회 -> 유저객체의 스캐쥴객체리스트

(범위일자)
토큰으로 유저 객체 조회 -> 해당 범위조회(스트림 이용) -> 유저객체의 스캐쥴객체리스트

(특별일자)
[개인일정]
토큰으로 유저 객체 조회 -> 유저 UserSchedule 객체 리스트 -> 해당 일자조회  -> 유저객체의 스캐쥴객체 -> 개인일정
[공고일정]
토큰으로 유저 객체 조회 -> 해당 일자조회  -> 유저객체의 스캐쥴객체 -> 공고일정
[특별일정]
토큰으로 유저 객체 조회 -> 해당 일자조회  -> 유저객체의 스캐쥴객체 -> 특별일정
~~~



일정 추가

~~~
[개인일정]
유저가 일자,내용 -> 유저 UserSchedule 객체에서 해당 일자 객체 조회 -> 있으면 개인일정 추가 , 없으면 생성후 개인일정 추가 

[공고일정]
유저가 공고선택 -> 일자, 공고 내용 ->  유저 UserSchedule 객체에서 해당 일자 객체 조회 -> 있으면 공고일정 추가 , 없으면 생성후 공고일정 추가

[특별일정]
유저가 특별일정 생성 -> 일자, 내용 ->  유저 UserSchedule 객체에서 해당 일자 객체 조회 -> 있으면 특별일정 추가 , 없으면 생성후 특별일정 추가
~~~



일정 수정

~~~
[개인일정]
유저가 해당 일자에 개인일정 탭을 수정 -> 일자, 개인일정 id, 수정내용 -> 유저 UserSchedule 객체에서 해당 일자 객체 조회 -> 해당 개인일정 조회 -> 내용 수정
~~~



일정 삭제

~~~
[개인일정]
유저 UserSchedule 객체에서 해당 일자 객체 조회 -> 일자, 개인일정 id, 수정내용 ->  해당 개인 일정 삭제

[공고일정]
유저가 해당 타겟 공고 삭제 -> 해당공고id 시작일 마감일 -> 유저 UserSchedule 객체에서 해당 일자 객체 조회 -> 공고일정에서 해당 공고 삭제 -> 해당 타겟 공고 일정 모두 삭제

[특별일정]
유저가 특별일정 데드라인 수정 -> 해당 특별일정 UserSchedule 객체에서 모두 삭제
~~~
