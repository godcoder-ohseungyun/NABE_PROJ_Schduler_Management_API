### 대웅(취업공고검색리스트결과)

사용자가 입력한 공고 검색 필터내용을 서버로 전송하면 그에 맞는 공고정보를 사람인으로부터 받아와서 필요한 정보만 추려서 제공한 리스트이다.

count와 total은 서버에서 필요한것으로 신경안써도 된다. 중요한건 job 리스트이다.

### 공통

각 객체들의 정보는 모두 앱에서 기억하고있어야한다. 그러나 사용자에게 보여지면 안되는 정보이다.

예를들어 사용자가 2020-05-04의 두번째 개인일정이라는 부분을 제거한다면 앱은 서버에 다음과 같이 제거요청을 해야한다.

id는 사용자에게 안보이여야하는 정보이지만 내부적으론 요청응답에 중요하게 사용되기 때문에 내부적으로 유지해야한다.

~~~json
{
    "targetDate" : "2020-05-02", //제거할 요일
    "targetId" : "1" //두번째 개인일정이라는 내용을 가진 객체의 id
}
~~~
